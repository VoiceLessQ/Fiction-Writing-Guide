# Fiction Writing Guide

A synthesised reference covering plot structure, subplots, 
unreliable narrators, show/don't tell, and dialogue craft.

## What is in here
- Story foundation (Nelson's 9 Steps)
- Plot structure (Freytag + 7-point)
- Plots and subplots
- Unreliable narrators (with psychology and illness angles)
- Show, don't tell
- Dialogue and subtext
